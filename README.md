# 🧁 Página de Receitas - Cupcake de Café

Uma página web dedicada a uma receita detalhada de Cupcake de café com chantilly. O projeto foca na organização de conteúdos textuais, listas de ingredientes e instruções de preparo de forma clara e atrativa.

> Projeto desenvolvido como parte do aprendizado na trilha da **Rocketseat**.

## 💻 Sobre o Projeto
O objetivo deste projeto foi trabalhar com a estruturação de conteúdos longos, utilizando tags semânticas para separar o resumo da receita, a lista de ingredientes e o passo a passo do modo de preparo, garantindo uma boa experiência de leitura.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica (main, section, ul, li).
* **CSS3:** Estilização de layout, tipografia e espaçamentos.
* **SVG:** Uso de ícones para detalhes visuais no rodapé.

## 🎨 Estrutura da Página
- **Destaque Visual:** Imagem principal do prato para atrair a atenção do usuário.
- **Resumo da Receita:** Informações rápidas como tempo de preparo, rendimento e nível de dificuldade.
- **Seção de Ingredientes:** Organizada em lista para facilitar o check-list durante o preparo.
- **Modo de Preparo:** Descrição detalhada das etapas de execução, recheio e montagem.
- **Rodapé Personalizado:** Assinatura do desenvolvedor.

## 📁 Organização de Arquivos
- `index.html`: Estrutura principal da página.
- `style.css`: Arquivo de estilização do projeto.
- `/assets`: Pasta contendo a imagem do cupcake e ícones utilizados.
